# 🛒 Amazon Clone

A responsive Amazon Clone built using HTML, CSS, and JavaScript.


## 📌 Features

- Responsive Amazon-style UI
- Modern Navigation Bar
- Search Bar
- Hero Section Banner
- Product Cards
- Product Ratings
- Product Prices
- Shopping Cart Counter
- Dark Mode Toggle
- Back To Top Button
- Hero Image Slider
- Local Storage Support
- Mobile Responsive Design

---
## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
